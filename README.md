## Run Locally

Clone the project

```bash
  git clone https://github.com/harsh-kashyap0201/Spotify-clone.git
```

Go to the project directory

```bash
  cd Spotify-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
